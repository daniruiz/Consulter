========== INSTALACIÓN ==========
1- Copiar ROOT.war a la carpeta de aplicaciones de tomcat
2- Copiar la carpeta sqliteDB a la carpeta raiz '/' para que la aplicación funcione con la base de datos.
3- El archivo citas.csv es el que contiene los datos de prueba para la carga de citas. Si no se cargan el listado de citas se verá vacío.
